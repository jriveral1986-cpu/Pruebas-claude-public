Crear Usuario

Solo el administrador puede crear nuevas cuentas. El registro público está deshabilitado.

Gestión de Administradores

Ingresa el UID de un usuario (visible en Firebase Console → Authentication) para otorgarle o quitarle permisos de administrador. Los administradores inician sesión solo con email/contraseña en login.html?admin=1.

Acceso de Administrador

El acceso de administrador usa usuario y contraseña fijos — sin opción de Google OAuth.
URL de acceso: /pages/login.html?admin=1
Credenciales: administrador / admin

Setup inicial (una sola vez en Firebase Console):

  1. En Firebase Console → Authentication → Agregar usuario manualmente.
  2. Email: admin@previsionchile.local · Contraseña: Admin.2026!
  3. Copia el UID generado y pégalo arriba → Otorgar permisos admin.
